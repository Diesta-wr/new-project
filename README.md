## ✨ Особенности

### 🍎 **Boot Screen**
- Анимация загрузки с логотипом Apple
- Плавное появление интерфейса

### 📱 **Реалистичный iPhone 15 Pro**
- Рамки, кнопки, Dynamic Island
- Реалистичные тени и градиенты
- Эффект стекла (backdrop-filter)

### 🏠 **Домашний экран**
- 16 иконок приложений в сетке 4x4
- Док (Dock) с 4 приложениями в стиле iOS
- Приложение **"ТЗ"** — единственное рабочее приложение

### 👥 **Приложение "ТЗ"**
- Карточки пользователей с API
- Поиск по имени, email, городу
- Сортировка по имени или городу
- Два режима отображения: сетка и список

### 🔍 **Поиск и фильтрация**
- Мгновенная фильтрация по вводу
- Поиск по имени, email, городу
- Подсветка найденных совпадений

### 📊 **Сортировка**
- По имени (А-Я)
- По городу (А-Я)

### ⊞☰ **Два режима отображения**
- **Сетка (grid)** — для наглядного просмотра
- **Список (list)** — для детальной информации

### 🟢 **Онлайн статусы**
- Плавно меняются каждые 3-7 секунд
- Зеленая точка у аватара
- Анимация пульсации

### ⚠️ **Обработка ошибок**
- 30% шанс ошибки при загрузке (для демонстрации)
- Красивый экран ошибки с кнопкой повтора
- Элегантное сообщение об ошибке сети

### 🎨 **Tailwind CSS**
- Современный подход к стилям
- Адаптивная верстка
- Кастомные анимации

### 🔄 **Параллакс эффект**
- Фон реагирует на движение мыши
- Плавное смещение градиентных кругов

## 📋 Требования из ТЗ

| ✅ | Поле поиска для фильтрации |
